OCLC Connexion
=======

Tools for the OCLC Connexion client.

* `macros` -- macros to run in OCLC Connexion
* `DateRangeBatchSearch` -- a web tool for use with Connexion to
search for resources added to OCLC in a given date range

More coming soon!
